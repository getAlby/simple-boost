Example: No-Confetti

  • Custom Styles
  • Disable confetti
  • JavaScript Events
  • Fiat amounts
  • A basic example
  • Lightning Address Support
  • Support for any NWC wallet

By default there will be confetti when a user sends a payment. You can disable this with the no-confetti attribute.

HTML

<simple-boost
  no-confetti="1"
  class="hey"
  address="hello@getalby.com"
  amount="2100"
></simple-boost>
